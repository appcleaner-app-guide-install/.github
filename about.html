About this site

Sustainable Tech Toolkit is a demo website built to be fast, accessible, and offline-first. It showcases a few ideas:

  • Installable PWA with service worker caching.
  • Client-side full-site search (no server required).
  • Light/dark theming with local persistence.
  • Modular structure with separate CSS/JS/assets.

Use it as a starting point or adapt the design for your own project.
